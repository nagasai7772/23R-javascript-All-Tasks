### Problem 3
Given a string inputString', use the ternary operator to check if it is a palindrome. A string is
considered a palindrome if it reads the same forwards and backwards.
- ** Output :** ' "Palindrome"' if the string is a palindrome,
- '"Not a Palindrome" otherwise.



var str = "world";
var str1 = "";
for (i = str.length - 1; i >= 0; i--) {
  str1 += str[i];
}
console.log(str1);
if (str == str1) {
  console.log("It's a palindrome");
} else {
  console.log("it's not a palindrome");
}
